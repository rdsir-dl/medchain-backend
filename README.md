@[MedChain] Backend & Blockchain Hub

This repository contains the integrated MedChain API and ephemeral Hardhat blockchain node.

## Quick Start
1. 
pm install
2. 
pm run cloud-start (Starts Hardhat + Deploys Contracts + Starts API)

## Structure
- lockchain.js: ESM-powered blockchain interface.
- server.js: Express API (ESM).
- models/: Mongoose schemas.
- contracts/: Solidity smart contracts.
